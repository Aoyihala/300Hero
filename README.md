# 一款md风格的300英雄战绩查询软件
#### 开发进度(10%)
> - 数据本地持久化
> - 角色绑定
> - 战绩列表分页查询
> - 战绩角色切换
> - mvp和角色在当前场次的角色定位计算
> - 针对于加载缓慢添加的ip代理地址
#### 数据来源
300战绩查询报告官网
#### 计算公式
1.**MVP**计算是根据评分和参团率和总经济来计算的,战败方名称为**躺输**计算规则一致。  

2.**坑**和**划水**是直接按评分来计算的，最后一名即划水大师或坑神。  

3.**神队友**是只要单局助攻大于30个即可，战胜方的标记。
